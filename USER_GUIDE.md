# 3D 액션 생존 게임 – 사용자 가이드

## 1. 게임 소개

이 게임은 상단 시점의 **3D 액션 생존 게임**으로,  
플레이어가 계속해서 등장하는 적을 처치하면서 **최대한 오래 살아남아 최고 기록을 갱신하는 것**이 목표입니다.

- 장르: 3D 액션 / 로그라이크 형태의 스테이지 클리어
- 플랫폼: PC (Windows, 마우스 + 키보드)
- 개발 도구: Unity
- 핵심 특징
  - 웨이브 형식으로 몰려오는 적
  - 근접 무기 + 원거리 무기 + 수류탄
  - 전투 후 상점에서 무기, 탄약, 체력 구매
  - 최고 점수 저장 기능

---

## 2. 실행 방법

### 2.1 빌드된 실행 파일로 실행

1. 제공된 압축 파일을 원하는 폴더에 압축 해제합니다.
2. 폴더 안의 실행 파일  
   `3DActionSurvivor.exe` (또는 팀에서 지정한 실행 파일 이름)을 더블 클릭합니다.
3. 해상도와 그래픽 옵션을 선택한 후 `Play` 버튼을 눌러 게임을 시작합니다.

### 2.2 Unity 에디터에서 실행 (개발용)

1. Unity Hub에서 프로젝트를 열거나, `Unity` → `Open Project`로 해당 프로젝트 폴더를 엽니다.
2. `Scenes` 폴더 안의 `SampleScene`을 더블 클릭해 연 후,
3. 상단의 ▶(Play) 버튼을 눌러 게임을 실행합니다.

---

## 3. 조작 방법

기본 키 설정은 다음과 같습니다.  
(팀에서 Input Manager를 변경했다면 이 표에 맞게 수정해서 사용)

### 3.1 이동 및 카메라

| 동작        | 키보드/마우스 입력           |
|-------------|------------------------------|
| 전후좌우 이동 | `W`, `A`, `S`, `D`           |
| 걷기(느리게) | `Left Shift` (길게 누르기)   |
| 시점 회전   | 마우스 이동 + 마우스 포인터가 가리키는 방향으로 시선 전환 |
| 점프        | `Space`                      |
| 회피(구르기) | 이동 중 `Space`              |

※ 점프는 제자리에서만 가능하고, 이동 중에는 회피로 동작합니다.

### 3.2 공격 및 무기 관련 조작

| 동작              | 입력                                  |
|-------------------|---------------------------------------|
| 기본 공격(근접/총) | 마우스 왼쪽 버튼 (`Fire1`)             |
| 수류탄 투척       | 마우스 오른쪽 버튼 (`Fire2`)           |
| 재장전            | `R` (`Reload`)                        |
| 무기 1 교체       | `1` (`Swap1`)                         |
| 무기 2 교체       | `2` (`Swap2`)                         |
| 무기 3 교체       | `3` (`Swap3`)                         |

- **마우스 방향으로 조준 후 공격**하면, 플레이어가 해당 방향을 바라보며 공격합니다.
- 탄약이 없을 경우, 재장전하거나 상점에서 추가 탄약을 구매해야 합니다.

### 3.3 상호작용 및 상점

| 동작             | 입력               |
|------------------|--------------------|
| 아이템/무기 줍기 | `E` (`Interaction`) |
| 상점 들어가기    | 상점 근처에서 `E`   |
| 상점 나가기      | 상점 UI에서 나가기 버튼 클릭 또는 일정 거리 벗어나기 |

## 4. 게임 진행 방식

### 4.1 기본 흐름

1. **게임 시작**
   - 메뉴에서 `Start` 버튼을 누르면 게임 카메라로 전환되고 플레이어가 활성화됩니다.
2. **스테이지 돌입**
   - 시작 존(Start Zone)에서 상점을 이용한 뒤, 스테이지 시작 트리거를 밟으면 적이 스폰됩니다.
3. **전투**
   - 적을 처치하면 점수와 코인을 획득합니다.
   - 일정 수의 적을 처치하면 해당 스테이지가 종료됩니다.
4. **휴식/상점 구간**
   - 스테이지가 끝나면 다시 시작 존으로 이동합니다.
   - 무기/탄약/체력을 상점에서 구매할 수 있습니다.
5. **다음 스테이지로 진행**
   - 스테이지 번호가 1씩 증가하며, 난이도(적 수/종류)가 점점 올라갑니다.
   - 5 스테이지마다 보스가 등장합니다.
6. **게임 오버**
   - 플레이어 체력이 0이 되면 게임 오버 화면이 나타나며, 이번 점수와 최고 점수가 표시됩니다.
   - `Restart` 버튼으로 처음부터 다시 시작할 수 있습니다.

## 5. 화면 설명 (UI 구성)

### 5.1 상단/좌측 UI

- **점수(Score)**: 현재까지 획득한 총 점수
- **스테이지(Stage)**: 현재 진행 중인 스테이지 번호
- **플레이 타임(Time)**: 게임 시작 후 경과 시간

### 5.2 하단 UI

- **플레이어 체력**: `현재 체력 / 최대 체력`
- **탄약 수**: `현재 장전 탄약 / 보유 탄약`
- **코인 수**: 획득한 코인 표시 (숫자가 커질수록 천 단위 구분 표시)

### 5.3 무기 및 수류탄 표시

- 무기 1, 2, 3 아이콘: 획득한 무기만 아이콘이 밝게 표시됩니다.
- 수류탄 아이콘: 수류탄을 1개 이상 보유하면 아이콘이 활성화됩니다.

### 5.4 보스 체력 바

- 보스 등장 시, 화면 상단에 보스 체력 게이지가 나타납니다.
- 보스가 사라지면 체력 바도 함께 사라집니다.

## 6. 아이템 설명

적을 처치하거나 맵에 배치된 아이템을 통해 다음과 같은 효과를 얻을 수 있습니다.

| 아이템 타입 | 설명                                               |
|------------|----------------------------------------------------|
| 탄약(Ammo) | 보유 탄약이 증가합니다. 최대 탄약량을 초과할 수 없습니다. |
| 코인(Coin) | 상점에서 사용 가능한 코인을 획득합니다.              |
| 하트(Heart)| 플레이어 체력이 회복됩니다. 최대 체력을 넘을 수 없습니다. |
| 수류탄(Grenade) | 수류탄 개수가 증가합니다. 최대 보유 개수를 넘을 수 없습니다. |
| 무기(Weapon) | 새로운 무기를 획득하여 슬롯(1~3)에 장착할 수 있습니다. |

## 7. 사운드

플레이어와 무기는 다음과 같은 사운드를 재생합니다.

- 점프
- 이동(발소리)
- 피격
- 회피/구르기
- 무기 교체
- 무기별 공격 사운드 (망치, 권총, 기관단총 등)

게임 옵션에서 전체 볼륨을 조절하여 쾌적한 음량으로 플레이할 수 있습니다.

## 8. 자주 묻는 질문(FAQ)

### Q1. 캐릭터가 이동하지 않아요.
- 게임 창이 비활성화 되어 있거나, Caps Lock/한영키 때문에 입력이 꼬였을 수 있습니다.
- Unity 에디터에서 실행 중인 경우 Game View가 선택되어 있는지 확인합니다.

### Q2. 공격이 안 나가요.
- 탄약이 0인지 확인 후, `R` 키로 재장전을 시도합니다.
- 근접 무기는 탄약이 필요 없지만, 공격 애니메이션 중에는 다른 행동이 제한될 수 있습니다.

### Q3. 상점이 반응하지 않아요.
- 상점 오브젝트에 충분히 가까이 접근한 뒤, `E` 키를 눌러 상호작용해야 합니다.
- 전투 중에는 상점이 비활성화됩니다. (스테이지 휴식 구간에서만 사용 가능)
